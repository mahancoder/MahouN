**تاریخ**: ۲۸ بهمن ۱۴۰۴

---

## خلاصه اجرایی

برای آماده‌سازی کامل ماحون برای بازار ایران، نیاز به آموزش (Fine-tuning) سه مدل اصلی با داده‌های حقوقی ایران داریم:

| مدل | کاربرد | پارامترها | هزینه تخمینی |
|-----|--------|-----------|--------------|
| Qwen2.5 | محاسبات گراف | 20-32B | $15,000-25,000 |
| Granit4 | استنتاج علّی | 20-30B | $15,000-25,000 |
| LLM Interface | رابط کاربری | 20-70B | $20,000-40,000 |
| **جمع کل** | | | **$50,000-90,000** |

**زمان**: ۳-۶ ماه  
**تیم مورد نیاز**: ۳-۵ نفر متخصص ML

## ۲. محاسبه هزینه آموزش

### ۲.۱ مدل Qwen2.5 (محاسبات گراف)

**مشخصات**:
- پارامترها: ۲۰-۳۲ میلیارد
- کاربرد: Node embeddings, similarity, graph traversal
- داده مورد نیاز: ۵۰۰ میلیون توکن

**هزینه‌ها**:

#### الف) هزینه محاسباتی (GPU)
```
سخت‌افزار: 8x NVIDIA A100 (80GB)
مدت: ۲-۳ هفته
هزینه اجاره ابری:
- AWS/GCP: $25/ساعت × 8 GPU = $200/ساعت
- مدت: ۳۰۰-۵۰۰ ساعت
- جمع: $60,000-100,000

گزینه اقتصادی (RunPod, Lambda Labs):
- هزینه: $10-15/ساعت × 8 GPU = $80-120/ساعت
- مدت: ۳۰۰-۵۰۰ ساعت
- جمع: $24,000-60,000

گزینه محلی (خرید GPU):
- 8x A100: ~$80,000 (یکبار)
- برق و نگهداری: $2,000/ماه
- استهلاک: $5,000/پروژه
```

#### ب) هزینه داده (Data Preparation)
```
جمع‌آوری و پاکسازی: $5,000
برچسب‌گذاری (اگر نیاز باشد): $10,000
تولید داده سنتتیک: $5,000
─────────────────────────────────────
جمع: $20,000
```

#### ج) هزینه نیروی انسانی
```
ML Engineer (۲ نفر × ۲ ماه): $20,000
Data Engineer (۱ نفر × ۱ ماه): $5,000
─────────────────────────────────────
جمع: $25,000
```

**جمع کل Qwen2.5**: $15,000-25,000 (با گزینه اقتصادی)
