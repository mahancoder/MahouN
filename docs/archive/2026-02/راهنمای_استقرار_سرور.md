## گام ۹: راه‌اندازی با systemd (Production)

### ۹.۱ ایجاد Service File

```bash
sudo nano /etc/systemd/system/mahoun.service
```

محتوای فایل:

```ini
[Unit]
Description=MAHOUN Platform API Server
After=network.target

[Service]
Type=simple
User=haji
WorkingDirectory=/home/haji/mahoun-platform
Environment="PATH=/home/haji/mahoun-platform/venv/bin"
ExecStart=/home/haji/mahoun-platform/venv/bin/uvicorn api.main:app --host 0.0.0.0 --port 8000 --workers 4
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

**نکته**: `User` و `WorkingDirectory` را با مسیر خودتان تغییر دهید!

### ۹.۲ فعال‌سازی Service

```bash
# Reload systemd
sudo systemctl daemon-reload

# فعال‌سازی
sudo systemctl enable mahoun

# شروع
sudo systemctl start mahoun

# بررسی وضعیت
sudo systemctl status mahoun

# مشاهده لاگ‌ها
sudo journalctl -u mahoun -f
```

### ۹.۳ دستورات مفید

```bash
# توقف سرویس
sudo systemctl stop mahoun

# ری‌استارت
sudo systemctl restart mahoun

# غیرفعال‌سازی
sudo systemctl disable mahoun
```

## گام ۱۰: نصب و پیکربندی Nginx (اختیاری اما توصیه می‌شود)

### ۱۰.۱ نصب Nginx

```bash
sudo apt install -y nginx
```

### ۱۰.۲ پیکربندی

```bash
sudo nano /etc/nginx/sites-available/mahoun
```

محتوای فایل:

```nginx
server {
    listen 80;
    server_name your-domain.com;  # یا IP سرور

    client_max_body_size 100M;  # برای آپلود فایل‌های بزرگ

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # برای WebSocket (اگر نیاز باشد)
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        
        # Timeout برای training jobs طولانی
        proxy_read_timeout 3600s;
        proxy_connect_timeout 3600s;
        proxy_send_timeout 3600s;
    }
}
```

### ۱۰.۳ فعال‌سازی

```bash
# ایجاد symlink
sudo ln -s /etc/nginx/sites-available/mahoun /etc/nginx/sites-enabled/

# تست پیکربندی
sudo nginx -t

# ری‌استارت Nginx
sudo systemctl restart nginx

# فعال‌سازی autostart
sudo systemctl enable nginx
```

### ۱۱.۲ Build Frontend

```bash
cd ~/mahoun-platform/frontend

# نصب dependencies
npm install

# تنظیم API URL
nano .env.production
```

محتوای `.env.production`:

```bash
VITE_API_BASE_URL=http://server-ip:8000
# یا اگر Nginx دارید:
# VITE_API_BASE_URL=http://your-domain.com
```

```bash
# Build
npm run build

# فایل‌های build در dist/ ایجاد می‌شوند
```

### ۱۱.۳ سرو Frontend با Nginx

```bash
# کپی فایل‌های build
sudo mkdir -p /var/www/mahoun
sudo cp -r dist/* /var/www/mahoun/

# تنظیم مجوزها
sudo chown -R www-data:www-data /var/www/mahoun
```

ویرایش پیکربندی Nginx:

```bash
sudo nano /etc/nginx/sites-available/mahoun
```

اضافه کردن:

```nginx
server {
    listen 80;
    server_name your-domain.com;

    # Frontend
    location / {
        root /var/www/mahoun;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # Backend API
    location /api {
        proxy_pass http://127.0.0.1:8000;
        # ... (بقیه تنظیمات proxy)
    }
}
```

```bash
# ری‌استارت Nginx
sudo systemctl restart nginx
```

## عیب‌یابی (Troubleshooting)

### مشکل ۱: Backend اجرا نمی‌شود

```bash
# بررسی لاگ‌ها
sudo journalctl -u mahoun -n 50

# بررسی port
sudo netstat -tulpn | grep 8000

# تست دستی
source venv/bin/activate
python3 -m uvicorn api.main:app --host 0.0.0.0 --port 8000
```

### مشکل ۲: Import Error

```bash
# بررسی venv
which python3
# باید نشان دهد: /home/user/mahoun-platform/venv/bin/python3

# نصب مجدد dependencies
pip install -r requirements.txt --force-reinstall
```

### مشکل ۳: CUDA not available

```bash
# بررسی driver
nvidia-smi

# بررسی PyTorch
python3 -c "import torch; print(torch.cuda.is_available())"

# نصب مجدد PyTorch با CUDA
pip uninstall torch torchvision torchaudio
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

### مشکل ۴: Permission Denied

```bash
# تنظیم مجوزها
chmod -R 755 ~/mahoun-platform
chown -R $USER:$USER ~/mahoun-platform
```

### مشکل ۵: Port already in use

```bash
# پیدا کردن process
sudo lsof -i :8000

# کشتن process
sudo kill -9 <PID>
```
