## 🎯 ویژگی‌های منحصر‌به‌فرد (مشترک در هر دو)

### 1. **Chain of Thought (6 مرحله)**
```
1. درک مسئله     → فهم عمیق موضوع
2. جمع‌آوری ادله  → استخراج مستندات
3. تحلیل علّی     → روابط cause-effect
4. استدلال قانونی → کاربرد قوانین
5. ارزیابی شواهد  → قدرت اثبات
6. نتیجه‌گیری    → حکم نهایی
```

### 2. **Causal Bayesian Networks**
- مدل‌سازی روابط علّی
- محاسبه احتمال شرطی
- استنتاج پیشین و پسین
- تحلیل حساسیت

### 3. **Legal Knowledge Graph**
```
قوانین → مواد → احکام
   ↓       ↓       ↓
شرایط → نتایج → مجازات
   ↓       ↓       ↓
ادله  → استدلال → رأی
```

### 4. **Evidence Assessment**
- وزن‌دهی به ادله (0-1)
- تشخیص تناقضات
- اعتبارسنجی منابع
- ترکیب ادله (Dempster-Shafer)
